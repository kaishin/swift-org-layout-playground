Safe, fast, and expressive.

Swift makes it easy to write software that is incredibly fast and safe by design.

Our goals for Swift are ambitious: we want to make programming simple things easy, and difficult things possible.

  let names = ["Anna", "Alex", "Brian", "Jack"]
  for name in names {
      print("Hello, \(name)!")
  }
  // Hello, Anna!
  // Hello, Alex!
  // Hello, Brian!
  // Hello, Jack!

Safe

Opting for safety sometimes means Swift will feel strict, but we believe that clarity saves time in the long run.

Fast

Swift must be comparable to C-based languages in performance for most tasks. Performance must also be predictable and consistent

Expressive

Swift benefits from decades of advancement in computer science to offer syntax that is a joy to use, with modern features developers expect.
